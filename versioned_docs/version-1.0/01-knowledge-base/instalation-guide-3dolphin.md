**\# Instalation Guide 3dolphin**
**\## Operating System**
- CentOS 9 Stream / Ubuntu / Rhel
**\## Timezone Configuration**
- Run : \`timedatectl set-timezone Asia/Jakarta\`
**\## Turn Off Firewalld**
\`\`\`bashsystemctl disabled firewalldsystemctl stop firewalld\`\`\`
**\## sysctl configuration**
- Append this parameter to \`/etc/sysctl.conf\`
\`\`\`bashnet.ipv4.ip_local_port_range = 31000 65500fs.aio-max-nr = 1048576fs.file-max = 6815744kernel.shmall = 2097152kernel.shmmax = 8329226240kernel.shmmni = 4096kernel.sem = 250 32000 100 128net.core.rmem_default = 262144net.core.rmem_max = 4194304net.core.wmem_default = 262144net.core.wmem_max = 1048586net.ipv4.tcp_sack = 0vm.swappiness = 1net.ipv4.ip_forward=1\`\`\`
- Run : \`sysctl -p\`- Reboot Server : \`sudo reboot\`
